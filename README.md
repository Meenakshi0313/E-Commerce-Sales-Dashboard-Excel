## E-Commerce Sales Performance Dashboard


## 📊 Project Overview

This project features an interactive Excel dashboard designed to analyze ₹76.03M in sales data. The goal was to transform raw e-commerce records into actionable insights regarding revenue trends, regional performance, and operational efficiency for a retail business.



## 💡 Key Business Insights

**Revenue Peak**: April was the highest-performing month, peaking at ₹19.73M.


**Product Strategy**: The "Set," "Kurta," and "Western Dress" categories are the primary revenue drivers.


**Operational Health**: The platform maintained a 14.2% cancellation rate and an 85.8% completion rate.


**Regional Dominance**: Maharashtra and Karnataka emerged as the top-performing states by both order volume and revenue contribution.


**Average Transaction Value**: The business achieved an Average Order Value (AOV) of ₹589.5 across 129K total orders.



## 🛠️ Technical Skills Demonstrated

**Data Modeling**: Managed and cleaned a dataset of 129K+ rows using Power Pivot.

**Advanced DAX**: Engineered custom measures for KPIs, including Net Revenue, Completion Rate, and Cancellation Percentages.

**Interactive UI/UX**: Designed a professional "F-Pattern" layout with custom-themed slicers for State, Status, Category, and Month.

**Professional Distribution**: Optimized for executive reporting by configuring print areas and scaling for single-page PDF exports.
